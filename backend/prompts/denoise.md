## 必须去除

- 广告横幅、推广文案、赞助商内容
- 导航链接、面包屑、菜单项
- 版权声明、cookie 提示、隐私政策摘要
- 页脚模板（footer）、社交分享按钮、订阅表单
- 重复段落和空白填充内容
- JavaScript 残留代码、CSS 样式片段
- "返回顶部""查看更多"等无信息量的 CTA 按钮文字
- 页码、分页导航

## 必须保留

- 产品介绍、功能描述、服务说明
- 价格信息、定价表、套餐对比
- 更新日志、版本说明、发布说明
- 核心文案（hero 区文字、价值主张、差异化描述）
- 关键数字、统计指标（用户量、性能数据等）
- 产品名称、品牌名、技术术语
- 表格数据（功能对比表、规格表等）

## 正反例

### 正反例 1：SaaS 落地页

**保留**：
```
## 定价方案
- 基础版：¥99/月，包含 3 个项目
- 专业版：¥299/月，包含 20 个项目 + API 调用
```

**去除**：
```
[免费试用] [预约演示] [联系我们]
```

### 正反例 2：博客文章

**保留**：
```
我们新增了批量导出功能，支持 CSV 和 Excel 格式，单次最多导出 10000 条数据。
```

**去除**：
```
分享到：微信 | 微博 | Twitter
作者：张三 | 发布于 2024-01-15 | 阅读量 1234
```

### 正反例 3：文档站

**保留**：
```
## API 认证
所有请求需在 Header 中携带 `Authorization: Bearer <token>`
```

**去除**：
```
上一篇：快速开始    下一篇：错误码说明
编辑此页 | 报告问题
```

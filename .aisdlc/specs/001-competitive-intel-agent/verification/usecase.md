## TC-001: 创建监控任务并保存扩展字段

**优先级：** P0（关键）  
**类型：** 功能 | 集成  
**状态：** 通过  
**预估时间：** 10 分钟  
**创建日期：** 2026-07-08  
**最后更新：** 2026-07-08

---

### 目标

验证当前实现能够保存监控任务，并正确写入 `competitor_urls`、`self_product_doc_name`、`competitor_contexts`、`cron`、`feishu_webhook` 等字段。

---

### 前置条件

- [x] Django API 已启动或可通过 Django 测试客户端调用
- [x] 请求体包含至少 1 组竞品 `title/url`
- [x] 测试环境允许写入 SQLite
- [x] 当前实现版本包含任务扩展字段

---

### 测试步骤

1. **向 `/api/projects` 发送创建请求**

   - **预期：** `competitor_urls[0].title`、`self_product_doc_name`、`competitor_contexts[0].supplement_doc_name` 与输入一致

3. **检查数据库侧结果**
   - 输入：创建后的 `MonitorProject`
   - **预期：** 记录成功持久化，字段结构符合序列化约束

---

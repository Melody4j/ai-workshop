# 前端报告与任务管理优化

## 需求 1：任务报告详情页增加 HTML 在线预览，移除 PDF 预览

当前任务报告详情页缺少生成的报告 HTML 在线预览功能，同时 PDF 预览是多余的，需要删除掉。

## 需求 2：任务管理页面卡片支持直接启动/停用调度任务

当前任务管理页面的任务卡片需要点进去才能开启/停用调度任务，希望在卡片上直接提供启动/停用操作按钮，无需进入详情页。

## 需求 3：执行详情页移除 HTML 和 MD 文件路径

任务监控的执行详情页面中，HTML 页面和 MD 页面的文件路径字段是多余的，去掉这些文件路径的展示。

## 需求 4：优化 HTML 报告模板排版

生成的 HTML 报告页面排版不够精美，内容太空泛，需要重新设计排版并优化 HTML 模板，使报告更美观、信息密度更高。

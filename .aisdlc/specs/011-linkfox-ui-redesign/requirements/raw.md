# 重构范围

基于当前前端已有页面进行统一重构，重点覆盖：

- 仪表盘 `/cockpit`
- 项目列表 `/projects`
- 项目创建 / 编辑页 `/projects/new`、`/projects/:id/edit`
- 情报列表 `/monitoring`
- 情报详情 `/monitoring/:id`
- 全局导航、侧边栏、页面容器、卡片、表格、表单、按钮、标签、空状态等通用界面元素

# 目标

- 全面参考 LinkFox AI 官网首页的布局结构、视觉层级、颜色风格、留白节奏和品牌感
- 重构当前前端的页面编排方式，让信息密度、可读性和视觉质感明显提升
- 统一当前系统的色板、字体气质、按钮样式、卡片样式、边框、阴影和交互反馈
- 在不破坏现有业务功能、路由和数据接口的前提下，完成前端 UI 层重构

# 约束

- 这是一次前端 UI / UX 重构，不是功能需求重写
- 现有页面功能、接口调用、核心信息结构应尽量保持兼容
- 新设计应适配桌面端主场景，并兼顾常见窄屏尺寸
- 需要形成统一的设计语言，而不是只改某一个页面
